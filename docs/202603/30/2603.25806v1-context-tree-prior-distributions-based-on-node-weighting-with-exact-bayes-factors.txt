CONTEXT TREE PRIOR DISTRIBUTIONS BASED ON NODE WEIGHTING WITH EXACT BAYES FACTORS    

> THIAGO PAULICHEN AND VICTOR FREGUGLIA
> Abstract. Variable-length Markov chains (VLMCs) are a flexible class of higher-order Markov models that admit a natural representation as context trees. The Bayesian approach to VLMCs involves assigning prior distributions to both the tree structure and the transition probabilities, with inference performed by integrating over nuisance parameters. We propose a Bayesian framework for context trees based on a novel class of prior distributions on tree space. The approach extends recursive methods used to compute the marginal likelihood and identify a posterior mode tree, accommodating flexible weighting schemes corresponding to these prior distributions. The proposed method generalizes existing approaches, as this class includes, among others, distributions generated by branching processes. This flexibility enables the selection of priors based on desired properties. Our framework enables efficient posterior exploration and allows model comparison and hypothesis testing via Bayes factors for context trees. To show the power of our methods, we conducted a simulation study comparing different prior choices, along with procedures based on Bayes factors to select the maximal depth and perform model selection.

1. Introduction 

Variable-length Markov chains (VLMCs) are a parsimonious class of higher-order Markov chains in which the probability of the next symbol depends only on a finite suffix of the past, called a context. The set of all contexts can be represented by the leaves of a rooted tree, provided that no context is a proper suffix of another. The set of leaves, together with the associated transition probability vectors, is called a context tree. The main inferential problem of a VLMC is to estimate the set of contexts from the data. Rissanen (1983) introduced the Context Algorithm, a sequential procedure to infer the tree structure, which prunes the tree nodes by aggregating past states that are irrelevant for prediction. Later, Bühlmann and Wyner (1999) adopted a more rigorous statistical approach to context trees and showed the consistency of the method. The Bayesian perspective on VLMC models, on the other hand, is a relatively recent research topic. Some works that explicitly adopt this paradigm include Dimitrakakis (2010), Freguglia and Garcia (2023) and Papageorgiou and Kontoyiannis (2024). This perspective arises as a translation of the weighting strategies used in data compression to improve prediction. A classical example is the Context Tree Weighting (CTW) algorithm (Willems et al., 1995), which computes the marginal probability of a sequence by weighting over all context trees and probability vectors. In general, the Bayesian approach is characterized by treating both the unobserved pa-rameters, the tree structure, and the transition probability vectors, as random variables, assigning prior distributions to each of them, and performing inference by integrating over the nuisance parameters. A standard prior distribution for the probability vectors, given the tree, allows integration to be easily performed by conjugation. Thus, the prior distribution for the tree structure can be viewed as a form of weighting this integrated quantity over all context trees. In this sense, the existing approaches correspond to specific prior choices

A quantity of interest in this perspective is the evidence (or marginal likelihood) since, apart from allowing model comparison, it corresponds to the normalizing constant of the posterior distribution. The main challenge lies in the computation of this quantity, because it involves a summation over the set of all context trees, and this set grows at a doubly exponential rate with the maximal depth considered. Therefore, the effectiveness of the Bayesian approach requires either a massive computational effort using Monte Carlo methods (Freguglia and Garcia, 2023) or the use of specific prior distributions that enable an exact recursive computation of the evidence (Kontoyiannis et al., 2022). Nevertheless, only a few distributions on full rooted trees are known, and even fewer enable the exact recursive computation of the marginal likelihood when selected as priors. To the best of our knowledge, so far, the only proposed distributions with this property are those generated by branching processes. The CTW algorithm, for example, recursively calculates the weighted probability of a sequence, which corresponds to the evidence under a prior distribution characterized by a branching process with a fixed branching probability of 1/2 at all nodes. Generalizations of this procedure are proposed by Kontoyiannis et al. (2022), extending the result to dis-tributions with any fixed branching probability and providing a more complete Bayesian framework. Nakahara et al. (2022) introduced an even more general distribution that al-lows different branching probabilities for each node and also has the property mentioned above. In parallel, methods that generalize the Context Tree Maximizing (CTM) algorithm (Volf and Willems, 1995) have also been developed for these distributions. These procedures correspond to obtaining a maximum a posteriori (MAP) tree under the Bayesian perspec-tive. Therefore, considering distributions of this form as prior ensures both exact posterior distributions and methods for obtaining a posterior mode tree. A key point here is that the class of distributions generated by branching processes is quite restrictive, not including, for example, the uniform distribution over the tree space, which is a natural uninformative prior. In this work, we introduce a new class of distributions on trees which incorporates the aforementioned ones and whose elements, when considered as prior distribution, preserve both the recursive computation of the evidence and the method for determining a MAP tree. This class is more flexible, as it contains, for example, the uniform distribution as well as other distributions that may be useful for hypothesis testing involving tree structures. The extension of the recursive methods for this class can be seen as generalizations of the CTW and CTM algorithms, allowing other weighting schemes. We discuss possible choices of prior distributions and conduct a simulation study to com-pare them in different scenarios, including the distribution utilized in Kontoyiannis et al. (2022). In each model, we analyze the value of evidence and evaluate the discrepancy be-tween the MAP estimators and the true trees. Some distributions may perform better than others in both criteria, depending on the nature of the data source. Furthermore, from the fact that certain hypotheses can be expressed by choosing specific prior distributions that fit our framework and the marginal likelihood can be calculated exactly and efficiently, we can perform hypothesis tests using the Bayes Factor (Kass and Raftery, 1995). This leads to results such as the estimation of the maximal depth via depth tests and model selection. The paper is organized as follows. In Section 2, we present the definitions of Context Trees and VLMCs. Section 3 describes our Bayesian framework, introducing the new class of prior distributions for the tree structure, as well as the results that ensure the effectiveness 3

Note that although the Dirichlet hyper-parameter α is fixed across models, the Bayes factor can also be used to test different values of α.The quantity log 10 BF F,G (z) can be interpreted as a measure of the evidence provided by the data z in favor of the model F over model G. Then, the comparison can be viewed as the gain/loss in entropy when considering a weighting induced by one context-tree function relative to another. 4.1.3. Results and comments. In simulation scenario (a), only two models correctly identify the true tree via the MAP estimator at the smallest sample size ( n = 200 ). Particularly, both are specified with target 3-depth context-tree functions. For larger sample sizes, several models obtain the true tree, differing only in their posterior probabilities. According to the log-evidence metric model T 38 performs best in all cases. The differences among the other target 3-depth models, however, are small. This behavior is expected, as the resulting prior distribution πT 3 

> c

assigns higher probability to trees close to the perfect tree of depth 3. An important point is that, under the best model T 38 , the MAP estimate is, in some cases, not the true tree but rather the perfect tree of depth 3. Nevertheless, the true tree still attains high posterior probability, indicating that the posterior distribution is strongly concentrated on trees that explain the data well. In simulation scenario (b), more models identify the true tree via the MAP estimator, even at small sample sizes. Interestingly, the target 3-depth model is successful, whereas the CTW models are not. A case that deserves emphasis is the model T 42 at larger sample size ( n = 2500 ). We observe that the MAP estimate is far from the true tree. In fact, the selected tree is the perfect tree of depth 4, indicating that the prior probability assigned to trees close to it may lead the model to identify dependencies in the data that do not actually exist. With respect to the log-evidence metric, the 0-indicator model I0 is the best performer and is significantly superior to the others. This is expected since the resulting distribution 

πI0 assigns prior probability only to 0-renewing trees. Unlike in the first scenario, under model I0 the MAP estimate coincides with the true tree for all sample sizes. Overall, for both simulation scenarios, the l-depth indicator models ( D3 and D4) perform well across all sample sizes, outperforming even the CTW models both in recovering the true tree and in terms of log-evidence. This illustrates that selecting a uniform prior at the correct depth can be more effective than arbitrarily choosing a prior distribution. This also highlights the importance of correctly specifying the maximal depth L.The exponential models perform worse for smaller sample sizes under both criteria. For larger samples, however, the true tree attains the highest posterior probability among all models. This is due to the strong penalization induced by these models. Therefore, in large-sample applications where structural estimation is desired, exponential models Eα may be advantageous. The CTW models consistently identify the true tree for larger samples, but they underper-form relative to the other models in terms of the log-evidence. In particular, the difference between CTW models under both criteria, when changing the β parameters, is negligible. Finally, as n increases, all models correctly identify the true tree, indicating that with sufficient data the influence of the prior becomes secondary. Even models such as T 42 in the first case and T 32 in the second, which correspond to misspecified priors that mistake the depth by one unit, successfully recover the true tree for larger sample. 4.2. Application - Maximal depth selection. A fundamental issue in our approach is the choice of the maximal depth L. Indeed, as in other CTW-type methods, this choice plays a crucial role and directly affects the behavior of the model. For instance, Gao et al. (2008) briefly comment on this in the context of entropy estimation based on the CTW method. As TL grows doubly exponentially with L, even a unit increase in the maximal depth substantially enlarges the model space. This has direct computational implications, since our Bayesian methods require constructing the maximal tree τMAX and evaluating all its sub-models. Therefore, although larger values of L allow the model to capture longer-term dependencies, they increase computational complexity and may lead to overfitting. 19 

To address this issue, as an application of our methodology, we develop an algorithm for selecting the maximal depth based on the Bayes factor (13). The core idea is that, while keeping L fixed, different maximal depths can be incorporated into the model through the use of the l-depth indicator function Dl. Specifically, for any two depths 0 ≤ l < k ≤ L,we decide between the hypotheses τ ∈ T l and τ ∈ T k by computing the Bayes factor of Dl

Ensure: l;

The algorithm starts at the maximal depth L and proceeds by sequentially testing smaller depths. The reduction of the model space is carried out only when there is sufficient evidence in favor of a simpler structure. Particularly, the required level of evidence is determined by the threshold c.Regarding the choice of c: smaller values favor greater maximal depths, whereas larger values promote smaller depths. Specifically, as c decreases, more evidence is required to reduce the model space from Tl to Tk in each algorithm phase. A standard reference value is 

c = 0 , as the sign of log 10 BF Dl,D k (z) serves as a direct measure of whether the data support the hypothesis τ ∈ T l (positive) or τ ∈ T k (negative). Alternative values of c may be selected based on conventional reference thresholds for interpreting log 10 BF, such as ±1/2, ±1 and 

±2. Nevertheless, in cases with small sample sizes, smaller values of c may be preferred to avoid the premature exclusion of potentially suitable trees. 5. Conclusions 

We developed a new Bayesian approach for context trees by introducing a novel class of prior distributions for the tree structure based on context tree functions. We demonstrated properties of context-tree functions that allow them to be combined to obtain distributions with desired characteristics, for example, with restricted domains. Other results, concerning operations on context-tree functions, show that it is possible to extend known recursive procedures for both the exact computation of the marginal likelihood and the determination of a MAP tree to the weighting schemes corresponding to these new prior distributions. The proposed class can be seen as a generalization of the previous literature, since it in-cludes the distributions obtained from branching processes. Other examples of distributions in this class were also presented, such as the uniform distribution, distributions with strong penalization (exponential distributions), and distributions that favor specific context depths (target l-depth distributions). In this sense, the wide variety of elements in this class allows the choice of a specific prior distribution (or weighting scheme) based on the needs and nature of the data. To illustrate the relevance of using alternative prior distributions, we analyzed artificial datasets generated from two models. We compared several prior choices, including, in each scenario, a suitable distribution. As expected, the results showed that different prior speci-fications lead to different effects: some choices performed better in identifying the generator tree but performed worse in terms of evidence, and vice versa. The results also demonstrated that models with suitable priors perform better, mainly in small samples. Some other inter-esting facts obtained from the simulation are the following: the uniform distribution at the correct depth may perform better than some other choices; exponential weighting schemes are good for identifying the true tree for larger samples; and finally, as the sample size grows, the influence of the prior becomes secondary. A recurring issue in the Bayesian framework is the selection of the maximal depth. To address this, we proposed a sequential algorithm based on the Bayes factor for context trees. The procedure was shown to be efficient in the simulated data, correctly identifying the true depth of the trees, although this was not its primary focus. This demonstrates the consistency of the Bayes factor as a tool for testing tree depths, suggesting its potential for testing other hypotheses expressed by priors for the tree structure. We then extended this method to model comparison by selecting both maximal depth and prior choices simultane-ously. Given a list of competing models, it identifies the best model for the data in terms of evidence. Therefore, our new methodology enables an effective posterior analysis for a larger class of prior distributions, and allows both model comparison and hypothesis testing through the Bayes factor. 22

In particular, we present the distributions obtained by selecting specific context-tree functions, especially those presented in Table 1, and provide a brief discussion of their properties. (1) Uniform distribution. Choosing the unity function U yields 

πU (τ ) = 1

|T L|, τ ∈ T L.

This can be used as an uninformative prior. (2) l-depth indicator distribution. For 0 ≤ l < L , the l-depth indicator function Dl gives 

πDl (τ ) = 1{τ ∈ T l}|T l| , τ ∈ T L.

This is the uniform distribution over Tl. It is useful for restricting the tree space to trees with depth at most l.(3) a-renewal indicator distribution. For a ∈ A , the a-renewal indicator function Ia leads to 

πIa (τ ) = 1{τ ∈ T aL }|T aL | , τ ∈ T L.

This is the uniform distribution over T aL , that is, it assigns uniform probability only to a-renewing trees. As a prior distribution, it is useful for problems that exhibit a known pattern of renewal. (4) Exponential distribution. For α > 0, the exponential function Eα produces
